# Graphql Service

## GraphQL endpoint

```http request
POST https://HOSTNAME/v1/graphql
```

** Execute a GraphQL mutation. **

### Parameters

| Field Name | Type | Description | Default |
| --- | --- | --- | --- |
| query | object | The query or queries to execute. | {} |
